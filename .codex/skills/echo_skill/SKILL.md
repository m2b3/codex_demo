# Echo Skill

When instructed to use this skill:

1. Identify the text the user wants echoed.
2. Return only that text converted to uppercase.
3. Preserve punctuation, spacing, and line breaks as much as possible.
4. Do not add explanation, labels, quotation marks, or extra commentary.
